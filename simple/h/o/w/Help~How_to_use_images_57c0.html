[edit] How can I put an image on Wikipedia?

Please read Wikipedia:Image use policy for information putting images on Wikipedia.

[edit] How can I put that image on a page?

[edit] Normal images

You can make an image on the right side of the page, in the text, by writing this:

[[Image:Wiki.png|frame|right|This is a logo]]

This is what you can see:

This is a logo

You can put the image on the left by writing [[Image:Wiki.png|frame|left|This is a logo]].

If you do not want the image in the text, write [[Image:Wiki.png|frame|none|This is a logo]]

This is what you can see:

This is a logo

[edit] Making an image smaller

Sometimes the image is very big, too big for the page. You can make it smaller by writing the number of pixels, for example "0px":

[[Image:Wiki.png|none|50px]]

This is what you can see:

A good way to make a small image is to use a thumbnail. Here is an example:

This is a logo

The "thumbnail" picture shows people that there is a bigger picture on Wikipedia, too. To make a thumbnail, write "Thumb":

[[Image:Wiki.png|150px|thumb|none|This is a logo]]
