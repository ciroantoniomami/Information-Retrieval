Operating system

From Simple English Wikipedia, the free encyclopedia

An operating system is what is needed to run the programs on a computer. It makes the link between programs and hardware.

An operating system is a mixture of different parts: (these can change depending on the operating system)

  • Kernel
  • Computer program and software
  • Driver (computer)
