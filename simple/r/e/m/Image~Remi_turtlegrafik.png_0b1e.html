Description

Example Output of Logo (turtle graphic).

Logo source code to create this graphic:

to n_eck :ne :sz
repeat :ne [rt 360 / :ne fd :sz]
end

to mn_eck :ne :sz
repeat :ne [rt 360 / :ne n_eck :ne :sz]
end

Start Program with:

mn_eck 36 20
