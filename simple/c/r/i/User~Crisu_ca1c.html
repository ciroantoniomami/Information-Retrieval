[edit] Etymology

This name was created at random. I created it when I chose letters off of signs on the highway. The name was originally a name for a character I created. I wanted the character to appear in a video game called Chrono Trigger. But I now use the name for myself.

My character's full name is Crisu Clef Skeer. The middle name is from an anime series called Magic Knight Rayearth. The last name is also a random creation.
