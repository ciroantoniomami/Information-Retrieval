Notepad

From Simple English Wikipedia, the free encyclopedia

Notepad is a basic word processing program that was created by Microsoft. It is very simple yet is still very effective. It has options such as placing a *"time stamp" into the document as well as changing the font, the font size, and the font style. The most common use for Notepad is to view or edit text (.txt) files, but many users find Notepad a simple tool for creating Web pages. Notepad can create website, yet requires a "code" to work (HTML). You can see an example of this in any web browser. To see the example, find your browsers option that allows you to view the source — in some cases, it will be under View >> Page Source or something similar.

  • Time Stamp - A stamp that gives the time and date.

A notepad is also the name given to several sheets of paper glued or put together in some way so one can flip between sheets.
