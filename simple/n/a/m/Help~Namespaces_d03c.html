Help:Namespaces

From Simple English Wikipedia, the free encyclopedia

Namespaces are the types of pages that you can find on Wikipedia. These help seperate the guidelines from the help documents, for example.

[edit] Main Namespace

The most common namespace on Wikipedia is the main namespace. All articles that do not have any prefixes before their name will be part of this namespace.

Examples of pages in this namespace are the Main Page and Music.

[edit] Talk Namespace

The Talk Namespace is used for discussion about an article. A talk page is made for every article that is made.

They are always prefixed with Talk:.

Examples of pages in this namespace are Talk:Main Page and Talk:Music.

[edit] Wikipedia Namespace

The Wikipedia namespace contains articles that are related to Wikipedia. These are often guidelines, project pages or community pages.

They are always prefixed with Wikipedia:.

Examples of pages in this namespace are the Wikipedia:Community Portal and Wikipedia:Esperanza.

[edit] User Namespace

The User namespace contains pages for each user on Wikipedia.

They are always prefixed with User:.

Examples of pages in this namespace are User:Jordanhatch and User:Archer7.

[edit] User Talk Namespace

The User Talk namespace contains talk pages for each user.

They are always prefixed with User talk:.

Examples of pages in this namespace are User talk:Jordanhatch and User talk:Archer7.

[edit] Help Namespace

The Help namespace contains help articles for members of Wikipedia.

They are always prefixed with Help:. A list of all help pages can be found at Help:Contents.

Examples of pages in this namespace are Help:Minor edit and Help:How to edit.

[edit] Special Namespace

The Special namespace contains scripts and non-editable pages of the encyclopedia.

They are always prefixed with Special:.

The most common page in this namespace is Special:Userlogin.
