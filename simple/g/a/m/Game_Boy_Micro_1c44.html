Game Boy Micro

From Simple English Wikipedia, the free encyclopedia

Nintendo Game Boy Micro

Game Boy Micro is a new video games handheld by Nintendo. It is another Game Boy Advance remake (like the Game Boy Advance SP) but with improved features from both of the ones before. For example, there is now a better light behind the screen which is even better than the Game Boy Advance SP and is probably the same as the Nintendo DS. It is slightly bigger than the iPod mini.

[edit] Release dates

It was released on the following dates:

  • September 13 2005 (JP)
  • September 19 2005 (NA)
  • November 4 2005 (EU)
