Game Boys

From Simple English Wikipedia, the free encyclopedia

The Game Boys are a group of handheld video game consoles made by Nintendo.

[edit] Game Boy

Main article: Game Boy

The first Game Boy was released in 1989. It had a small screen, a D-pad, and an A and B button. To play games, there were little things called cartridges that you put into the Game Boy to play the game in the cartridge. The game that made it really good was called Tetris. The Game Boy had a link cable and a link port which let it connect with other Game Boys.

[edit] Game Boy Pocket

After the Game Boy, Nintendo released the Game Boy Pocket. It did not need as many batteries as the Game Boy. The link port was smaller than the Game Boy's, so the Game Boy and Game Boy Pocket could not connect.

[edit] Game Boy Light

The Game Boy Light was only in Japan and had a backlight so people could see better. It is very rare. Because it is rare, it sells for more money than other Game Boys.

[edit] Game Boy Color

The Game Boy Color had more color than the Game Boy Pocket. It was also able to play Game Boy cartridges, which helped it sell better. This also helped other Game Boy models after the Game Boy Color. It is somtimes called GBC.

[edit] Game Boy Advance

Main article: Game Boy Advance

The Game Boy Advance is a handheld that had better color and smaller cartridges. It is sometimes called GBA. It came out in 2001.

[edit] Game Boy Advance SP

Main article: Game Boy Advance SP

The Game Boy Advance SP was an improved version of the GBA. It has a clamshell design, which means you can fold it to make it smaller. It has a better light. It can play Game Boy and Game Boy Color games. There is now a new kind of Game Boy Advance SP with a light call

[edit] Nintendo DS

The DS is not a Game Boy, but it goes with the Game Boy. It has two screens, one of them is a touch screen. DS games use cards, but it also accepts cartridges. It cannot use Game Boy and Game Boy color games.

[edit] Game Boy Micro

Main article: Game Boy Micro

The Game Boy Micro is a smaller GBA. It has a smaller link port than the Game Boy Advance. It has the very best screen out of all the Game Boys.
