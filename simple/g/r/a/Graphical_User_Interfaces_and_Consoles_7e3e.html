Graphical User Interfaces and Consoles

From Simple English Wikipedia, the free encyclopedia

Computers can display information and let the user give commands to it using two methods: a Console or a Graphical User Interface (GUI).

In a Console, the user types commands using the keyboard to tell the computer to take an action. For example, the more command available in most operating systems will display the contents of a file.

In a Graphical User Interface, the user can use the computer mouse to click on buttons. For example, your web browser probably has a Print icon to print this page.

Graphical User Interfaces are generally simpler to use than Consoles, whereas Consoles are faster than Graphical User Interfaces and can be used to give special commands to the computer.

List of Operating Systems with Graphical User Intefaces

  • Microsoft Windows
  • Mac OS
  • Linux
