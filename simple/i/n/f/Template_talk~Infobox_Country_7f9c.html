[edit] Usage guidelines

This is still new, but here is a usage guide for the template

  1. Copy the text below and paste it into the country article.
  2. Edit the parameters with the correct information.
    • nationalmotto & nationalsong - use quotes

Please try to use English translations for any text displayed, even if the article page resides somewhere else.

From the Germany page (only an example):
