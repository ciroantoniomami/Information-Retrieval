Template:Infobox actor/doc

From Simple English Wikipedia, the free encyclopedia

 This is the template documentation page for {{Infobox actor/doc}}.
It is not intended to be viewed directly. If it uses variables, some links may appear broken. Do not replace these variables with hardcoded page names or URLs.

An Infobox actor may be used to summarize information about a person who is an actor/actress.

[edit] Usage

The infobox may be added by pasting the template as shown below into an article. Only the name and birthdate parameters are required. For an example, see Helen Hunt.

{{Infobox actor
| bgcolour      = 
| name          = 
| image         = 
| imagesize     = 
| caption       = 
| birthname     = 
| birthdate     = 
| location      = 
| deathdate     = 
| deathplace    = 
| othername     = 
| yearsactive   = 
| spouse        = 
| homepage      = 
| notable role  = 
| academyawards = 
| emmyawards    = 
| tonyawards    = 
| imdbid        =
}}

[edit] Parameters

Note: * denotes required fields.

Parameter Explanation
bgcolour Insert the background color. This should typically be left blank.
name * Insert name of actor. Use the common name (typically name of article) if multiple names are used.
image Insert *.jpg image.
imagesize Insert image width (defaults to 220px if empty).
caption Insert photograph caption. Try to include date of photo and the photographer.
birthname Insert actor's name at birth (or christening if name at birth is unavailable).
birthdate * Insert date of birth: month, day, year. Consider using {{birth date and age}} for a living person.
location Insert place of birth: town, city, country.
deathdate Insert date of death: month, day, year.
deathplace Insert place of death: town, city, country.
othername Insert other name(s) that the actor has been known by.
yearsactive Insert the years that the actor was active in his/her career. Use the format:

[[year of first professional performance]]–[[year of final professional appearance]]

Note: If still active, use 'Present' in place of the end year.
spouse Insert the names of the actor's spouses. Use the format:

Name (Year–Year)

Note: If still married, use 'Present' in place of the end year. In addition, separate multiple entries with a line break (<br />).
homepage Insert the actor's official website.
notable role Insert the role(s) for which the actor is best known. Use the format:

RoleName in FilmName
academyawards Insert any Academy Awards won. Use the format:

[[Award Name's Article|Short Form of Award Name]]
Year Title (optional: – Role)

Note: If an article for the film is available, the title should link there.
emmyawards Insert any Emmy Awards won. Use the same format as academyawards.
tonyawards Insert any Tony Awards won. Use the same format as academyawards.
goldenglobeawards Insert any Golden Globe Awards won. Use the same format as academyawards.
baftaawards Insert any BAFTA Awards won. Use the same format as academyawards.
cesarawards Insert any César Awards won. Use the same format as academyawards.
goyaawards Insert any Goya Awards won. Use the same format as academyawards.
afiawards Insert any AFI Awards won. Use the same format as academyawards.
filmfareawards Insert any Filmfare Awards won. Use the same format as academyawards.
olivierawards Insert any Laurence Olivier Awards won. Use the same format as academyawards.
imdbid Insert the IMDb profile number for the actor.
