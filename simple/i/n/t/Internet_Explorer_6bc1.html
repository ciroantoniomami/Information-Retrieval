Internet Explorer

From Simple English Wikipedia, the free encyclopedia

Internet Explorer is a web browser created by Spyglass Inc. It was licensed by Microsoft in 1996 and has been developed by them ever since. In 1996, Microsoft bought Internet Explorer, which was called Mosaic at the time, from Spyglass Inc and gave them some of the money they made from it. The browser has been developed by Microsoft ever since.

Internet Explorer has been included in every version of the Microsoft Windows operating system since Windows 98, in 1999. Microsoft used to make versions of the browser for UNIX, Solaris, and Macintosh operating systems.

Many people who use other browsers do not like Internet Explorer. They say it is insecure and has lots of problems in it, for example, holes in its security.
