Template talk:Imdb title

From Simple English Wikipedia, the free encyclopedia

This is a template to generate links to the Internet Movie Database. This template should be used exclusively for TITLES, that is, movies, TV shows, games, etc.

If you need to link a person (actor, actress, etc.), use Template:imdb name instead.

[edit] Instructions

  1. Go to imdb.com and search for the title you want
  2. Click on the correct link to go to the movie's page (or whatever you're looking for). Be sure you're getting on the correct title page.
  3. The page URL should look like http://www.imdb.com/title/tt#######, where "#######" indicates a number.
  4. Copy ONLY that number (be sure to remove the "tt" and to keep all the zeros)
  5. Add to the external links section the text:
{{imdb title|id=#######|title=Title}}
where "#######" is the number you've copied, and "Title" is the title of the movie or show.

Note: If the title is not specified, the name of the article is used as the title.

[edit] Example

  1. Searching for "Beetle Juice", you'll get to this page: http://www.imdb.com/title/tt0094721/
  2. Copy "0094721" and then write:
{{imdb title|id=0094721|title=Beetle Juice}}

The result is Beetle Juice at the Internet Movie Database (IMDb)

To ease the server load, even better is to 'subst' the template by typing:

{{subst:imdb title|id=0094721|title=Beetle Juice}}

The result will also appear:

Beetle Juice at the Internet Movie Database (IMDb)

But if you edit the page again you will see it has been converted to:

[http://www.imdb.com/title/tt0094721/ ''Beetle Juice''] at the [[Internet Movie Database]]

Note that, since this will almost certainly go in the "External links" section, you should add a * before the template text to include it as a list item.
