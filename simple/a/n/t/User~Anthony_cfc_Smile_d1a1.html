[edit] Usage

To place this template on a page, type {{subst:user:anthony cfc/Smile}}; this will place the above box on the page. Remember to place it on user talk pages only - not article, or article talk pages..or userpages, for that matter, unless the user has a notice saying it's okay. To go to a user's talk page, click the "talk" page at the top of the userpage.
