[edit] From the Basic English institute

There are two bits of open software and the Institute provides two word lists to replace those provided with each software.

[edit] Simple English word check

OpenOffice.org (OOo) is a full clone of Microsoft Office Professional -- and it is a free download (55MB) or is available on CD. The Institute has a wordlist download

Replace the one of the number of languages of OOo spell checker with the Basic English wordlist of 5,000+ spellings. Non-Basic words are highlighted as if misspelled.

[edit] Translation from standard English into Simple English

Internet Dictionary Project software called IDP Companion translates between several languages. Replace one of the minor languages with the Basic English definitions from the Institute download page. Enter the "misspelled" word and copy and paste the desired Basic replacement.

Simple English is just an impure form of Basic. Therefore, if you are not able to easily translate into Basic, the result is called Simple English.

[edit] Write Idea

WriteIdea is an Open Source writers' tool for simplifying English text. There is a java based version you can download, and a web-based version you can use online.
