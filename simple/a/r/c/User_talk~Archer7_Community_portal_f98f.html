[edit] About this page

I created this page because I think the Community portal needs to be a bit more inviting to new users and include much more, such as the sandbox and peer review. I've tried to keep the basic structure (expand, create etc) the same, but I've added help and community sections and our version of a collaboration to improve a certain category of articles (in this case Chemistry). Please let me know what you think. Archer7 20:41, 1 February 2006 (UTC)

I tweaked it a bit to get rid of some of the blank space that didn't really seem necessary, but increased the vertical scrolling needed to navigate the page--I would have also reduced the blank space above the four main headers if I could have figured out where it was coming from. Also, I've seen many people complain about blocks of yellow, so I borrowed more of a cream color from a section of the EN version. If people feel it's too pale, maybe the tan (#FFEEDD) of EN's 'writing resources' section would work. Of course, I realize while I think the changes I made are an improvement, I know if others don't agree it's easy to undo them. The main work has already been done very successfully by Archer7, with a boost from Aflm.

I do think that this is a good time to re-evaluate using the word "portal" for this page, as I believe it isn't very simple, and doubt it would seem as correct/necessary if EN didn't use it so much in so many contexts. I think something like 'Community gateway' or 'Community entry' would convey the same general sense, but with Simpler wording. Freshstart 23:38, 13 February 2006 (UTC)
