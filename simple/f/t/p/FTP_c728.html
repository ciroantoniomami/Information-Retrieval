FTP

From Simple English Wikipedia, the free encyclopedia

FTP stands for File Transfer Protocol. FTP is a way to send and receive files on the internet.

FTP is a way of connecting to another computer (called a server) to access its files. The computer that connects to the server is called a client. The client needs special FTP software to connect to the server. This is how most information is put on the Internet.

The server listens on the network (a system for connecting computers) for connection requests from other computers. The client can make a connection to the FTP server by using the special software. Once connected, the client can do things such as uploading files (putting its own files on the server) to the server, download files (taking the servers files and putting them on his own computer) from the server, rename or delete files on the server and so on. Any software company or individual programmer is able to create FTP server or client software because the protocol is an open standard. Nearly all computer platforms support the FTP protocol. This allows any computer to connected to a TCP/IP based network to manipulate files on another computer on that network regardless of which operating systems are involved (if the computers permit FTP access).

There are many existing FTP client and server programs, and many of these are free.
