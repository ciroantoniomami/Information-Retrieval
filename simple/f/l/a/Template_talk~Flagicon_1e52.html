[edit] Unsupported Countries

The template will work for any country that has an image of its flag in the format of Flag of Country.svg. The country name must be spelled the same as it is in the image name. Short names can not be used unless an image exists with the shortened name. Countries with images of flags that are not .svg files (.jgp or .png files) also will not work. These countries must have alias templates created for them to work properly.

[edit] Creation of alias templates for other countries

Alias templates need to be created for each country that is to be used. These templates define the official name of the country, the name of the image to be used as its flag and any short versions of the name of the country. At least 2 alias template need to be created for each country. An optional alias template for each short version of the country name to be used must also be created for it to work. Templates that need to be created are:

  1. Template for country name: Template:Country alias Country name
    This template has one single line that contains only the full name of the country. ie. United States
  2. Template(s) for short name(s) of the country. ie. Template:Country alias USA (optional)
    This template redirects to the main country name template
  3. Template for the country flag: Template:Country flag alias Country name
    This template contains the name of the image file for the flag. ie. Flag of the United States.png
