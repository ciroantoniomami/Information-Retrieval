Ubuntu

From Simple English Wikipedia, the free encyclopedia

Ubuntu is a Linux operating system. It is one of the most popular types of Linux. An idea behind Ubuntu is to make it easy to use.

These are the three types of free CDs.

This means it has been made so that is easy to put onto a computer and to use it. The word "ubuntu" is an old African word: it means "humanity".

Ubuntu can be used on all types of systems like Macs and PCs. Ubuntu is put on computers by using a CD, which are available for free on the Ubuntu website.

Started in 2004 by a wealthy South African man by the name of Mark Shuttlesworth, Ubuntu has been made at a fast rate. Because there are several options for which desktop software to use, Ubuntu was split into four different packages: Kubuntu, Edubuntu, Xubuntu, and Ubuntu.
