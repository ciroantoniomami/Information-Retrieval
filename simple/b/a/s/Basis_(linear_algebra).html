Basis (linear algebra)

From Simple English Wikipedia, the free encyclopedia

In Linear algebra, a 'Basis is a set of vectors with certain propeties:

  • Using Addition and Multiplication, it is possible to reach any point in a given Vector space.
  • There is no Vector that can be left out, so that the property above is still satisfied.
