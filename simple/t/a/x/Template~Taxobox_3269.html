[edit] About this template

This template sets up a taxobox. Although the implementation is complicated, it is relatively simple to use. See taxobox usage for an explanation. For an example in practice, look at the edit page of eel - [1]. Most parameters are optional, so if a particular entry is not relevant for your case, just leave it out.

This template contains one or more optional parameters: see the talk page for details.

This template employs some extremely complicated and esoteric features of template syntax. Please do not attempt to alter it unless you are certain that you understand the setup and are prepared to repair any consequent collateral damage if the results are unexpected. Any experiments should be conducted in your user space.
