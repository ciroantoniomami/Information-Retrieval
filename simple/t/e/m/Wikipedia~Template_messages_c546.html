Wikipedia:Template messages

From Simple English Wikipedia, the free encyclopedia

This page is a guideline on Wikipedia. Many editors agree with acting this way. It is a good idea to follow it, but it is not policy. Feel free to change the page as needed, but please use the talk page to suggest any big changes.
Shortcut:
WP:TM
WP:TEMP

Templates are used within pages to provide the same message over and over again. Changes to the look, wording, or style of the message can be done quickly, because templates are made again for the reader when the article loads.

In other words, when the message is changed, you do not need to update a page that uses it if you use a template. Standard notices that might be on a page for months or forever should be "transcluded" for easy updating. This loads the current version of the message and puts it on the page every time you load it.

For any of these messages, you can include "subst:" (as in {{subst:1911}} ) when inserting them to put the full text into the page. Doing this means that any future changes to the template will not be shown in that article, but it can be useful sometimes.

To make a new custom message, start a page in the Template namespace. The title should be "Template:", then the name of your message. The page should then be written with the text of your message. Add your new template to this list.

[edit] Delete an article

The templates here are used to let administrators know about articles you think should be deleted. See Wikipedia:Deletion_policy for when to use these.

Name Current text Where
{{pdel|reason}}
(talk)

Somebody has nominated this project page for deletion. Their reason for this is: give a reason

You may join the discussion if you'd like to give your opinion on this request. Please do not remove this notice or empty this page while the discussion is happening. You are welcome to edit this page and improve it. If you want to find out more information, please read Wikipedia's deletion policy.
 Articles
top
{{delete|reason}}
(talk)
An editor has asked for quick deletion of this page. The editor's reason for asking for deletion is: give a reason

If you disagree with its deletion, please explain why on this page's talk page. Please do not remove this notice from pages that you have created yourself.

Administrators, remember to check what links here, the page history (last edit), and the page log, before deletion.
 Articles
top
{{non}}
(talk)
An editor has asked for quick deletion of this project page. The editor's reason for asking for deletion is: It doesn't make any sense (It's obvious nonsense). (QD G1)

If you think this page should not be deleted, please say why on this page's talk page. Please do not remove this notice from pages that you have created yourself.

Administrators, remember to check what links here, the page history (last edit), and the page log, before deletion

{{notable}}
(talk)
An editor has asked for quick deletion of this project page. The editor's reason for asking for deletion is: The article does not explain why the subject is notable. (QD A4)

If you think this page should not be deleted, please say why on this page's talk page. Please do not remove this notice from pages that you have created yourself.
