Microsoft Office

From Simple English Wikipedia, the free encyclopedia

Microsoft Office is a group of computer programs created by Microsoft which lets you make documents and send e-mails. It is designed to be the single, most important program an office needs to work. Another program which tries to do the same is the free and open source "OpenOffice".

[edit] Main Programs

There are four common programs in Microsoft Office:

[edit] Microsoft Word

A word processing program which lets you use different fonts and colours to your text. You can also add images, clip art and diagrams.

[edit] Microsoft Excel

A spreadsheet program which lets you work with numbers, calculate equations and write formulas.

[edit] Microsoft PowerPoint

Creates and edits presentations for slideshows, meetings and webpages

[edit] Microsoft Outlook/Entourage

Microsoft Outlook is a program in which it is used for common emails. Outlook is only for Windows users. Microsoft Entourage is an email program created for Apple Macintosh OS X. It is not a Windows program.
