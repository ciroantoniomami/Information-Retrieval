Microsoft Windows

From Simple English Wikipedia, the free encyclopedia

Microsoft Windows is an operating system made by Microsoft. It is currently the most popular operating system in use. It is installed on most new computers.

Microsoft Windows was first made in 1985. The most recent version is Windows Vista which was made in 2007.

A man named Bill Gates made Windows. He is now the vice chief executive officer of Microsoft.

Windows lets you run programs and sort your files. It has many programs that come with it, these can read and write simple documents and pictures. There are also applications which let you use the internet, looking at web pages and reading e-mail. There is also an application for listening to music and watching videos, it is called Windows Media Player.

Some people like Windows because it is easy to use and to learn. It also works very well on many different parts of computer hardware.

Some people do not like it. They say it can break and stop working often. They also think it is too big and uses a lot of computer memory.
