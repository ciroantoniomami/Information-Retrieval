Windows Media Player

From Simple English Wikipedia, the free encyclopedia

Windows Media Player is a media player made by Microsoft. It is already installed on Microsoft Windows operating systems, and a different version is available for some Apple Macintosh operating systems. The program allows people to watch certain video files and play music files that are in a compatible format. How the program looks can also be changed with skins, and it can be made small enough to fit on the taskbar.
