Windows NT

Windows NT was the first of Microsoft's operating systems to use the new 'NT' (New Technology) kernel, which is still partly present in Windows XP (NT 5.1). A kernel is a file which allowed the computer to run from the processor, making the computer work. That meant it had a brand new kernel to do more tasks than the MS-DOS based kernel. The new kernel meant that Windows was more secure and crashed less.
