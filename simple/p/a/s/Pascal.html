Pascal

From Simple English Wikipedia, the free encyclopedia

The mathematician can be found at Blaise Pascal

Pascal is a programming language. It is created in 1970 by Niklaus Wirth. In the beginning pascal was a language made to learn how you can make good programs.

Unfortunately there are different dialects of the language. So a Pascal program written for one compiler can not easily be used with another one.

[edit] Pascal variants

  • GNU Pascal
  • Free Pascal
  • Delphi
  • Turbo Pascal
