Template:Succession box/doc

From Simple English Wikipedia, the free encyclopedia

 This is the template documentation page for {{Succession box/doc}}.
It is not intended to be viewed directly. If it uses variables, some links may appear broken. Do not replace these variables with hardcoded page names or URLs.

This template is used to show the previous and the next in a row of things.

[edit] Usage

[edit] Single box

{{start box}}
{{succession box | title=King of the Hill | before=Bob the Great | after=Peter the Nimble | years=1964–1970}}
{{end box}}

This produces:

Came after:
Bob the Great		 King of the Hill
1964–1970		 Came before:
Peter the Nimble

[edit] More rows

For more rows, just add more {{succession box}} tags.

{{start box}}
{{succession box | title=King of the Hill | before=Bob the Great | after=Peter the Nimble | years=1964–1970}}
{{succession box | title=Mister Universe | before=John the Pretty | after=Roger the Beautiful | years=1950}}
{{end box}}

This makes:

Came after:
Bob the Great		 King of the Hill
1964–1970		 Came before:
Peter the Nimble
Came after:
John the Pretty		 Mister Universe
1950		 Came before:
Roger the Beautiful
