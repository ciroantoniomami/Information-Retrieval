[edit] Interwiki link

Shenme, to do a visible interwiki like the one on your user page, simply add colon to the front of the link. So, writing [[:commons:|Commons Main Page]] will create this link: Commons Main Page. You can create links to any page in any wiki in this fashion. For another example, my user page has a link to my user page at English Wikipedia. The colon after the brackets also work for categories (if you're refering to a category but don't want to make the page you are in a part of that category) and to images (a link to the image versus simply showing the image). --Ricky81682 (talk) 04:54, 21 May 2005 (UTC)

It's not really a problem on User:SqueakBox's user page. Well, now he's changed it. Let me try this for example. The link [[:simple:User talk:Shenme]] shows up as simple:User talk:Shenme because simple:User talk:Shenme refers this page that I'm writing on. It's Shenme's user talk page in the Simple wiki.
